Политика конфиденциальности

Обновлено: 22.09.2025

Мы уважаем вашу приватность. Эта политика описывает, какие данные собирает и как использует сервис «AI Assistant для Telegram» на ai-assistant-tg.duckdns.org.

Какие данные мы обрабатываем

  • Данные Telegram: ваш user_id, username, сообщения боту.
  • Данные подписки и биллинга: тариф, срок действия, факты оплат.
  • Интеграции Google (по вашему разрешению): токены OAuth, идентификаторы и метаданные документов/календарей; содержимое используется только для выполнения запросов (поиск/чтение/запись), согласно выданным правам.
  • Технические журналы: время запроса, количество токенов ИИ-модели, диагностические ошибки.

Зачем мы это делаем

Для предоставления сервиса бота, подсчёта лимитов, генерации ответов ИИ, интеграции с Google и улучшения качества (отладка и аналитика).

Хранение и удаление

  • SQLite БД и кэш Redis находятся на наших серверах.
  • Токены OAuth хранятся в зашифрованном виде и могут быть отозваны вами в Google Account > Security > Third-party access.
  • По запросу через Telegram (@rebornerr) удалим ваши данные, кроме законодательно обязательных.

Передача третьим сторонам

Передаём только поставщикам услуг, необходимым для работы сервиса (например, OpenRouter для ИИ-моделей, YooKassa/Crypto для платежей). Содержимое ваших документов не используется для обучения моделей.

Контакты

Владелец сервиса: ИП/ФИО (укажите). Контакт: email/Telegram @rebornerr.
